# polo


#### 주요기능

1. 회원가입 로그인 이메일찾기
2. 비밀번호 정규화및 해쉬암호화
3. 게시판 글 등록,삭제
4. 상품등록 사진및 규칙에 맞게 상품등록 할수있는 틀
5. 쇼핑리스트 담기, 페이지 번호
6. 장바구니 토탈가격과 담은걸 삭제
7. 카카오 API 지도
8. 게시판 검색, 페이지 번호

이것은 쇼핑물 백엔드위주 페이지 입니다.
